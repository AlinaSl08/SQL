# SQL

**SQL (Structured Query Language)** – это язык структурированных запросов, с помощью которого мы **общаемся с реляционной базой данных**. SQL-команды делятся на группы — в зависимости от того, что они делают с **базой данных**.

 
**Основные группы SQL-команд:**

|Группа	|Название	|Что делает|
|:--:|:--:|:--:|
|**DDL**|	Data Definition Language|	Создаёт и меняет структуру базы|
|**DML**|	Data Manipulation Language|	Работает с данными внутри таблиц|
|**DQL**|	Data Query Language|	Получает данные (запросы)|
|**DCL**|	Data Control Language|	Управляет доступом|
|**TCL**|	Transaction Control Language|	Контролирует транзакции (подтвердить/отменить)|
 

**В этому курсе мы рассмотрим группы DML и DQL, так как они чаще всего используются в работе тестировщика.**

|Запрос|	Что делает|	Примеры|
|:--:|:--:|:--:|
|**SELECT (DQL)**|	Читает данные|	"Покажи все заказы"|
|**INSERT (DML)**|	Добавляет данные|	"Создай нового пользователя"|
|**UPDATE (DML)**|	Обновляет данные| 	"Измени цену товара"|
|**DELETE (DML)**|	Удаляет данные| 	"Удалить заказ"|
