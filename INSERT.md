# INSERT

Команда `INSERT` используется в SQL, чтобы добавить новую строку в таблицу – то есть сохранить данные в базе.

**Например: когда пользователь регистрируется, его данные добавляются в таблицу users (называние может быть и другое) – через SQL-команду INSERT**

**Синтаксис:**
``` SQL
INSERT INTO имя_таблицы (колонка1, колонка2, ...)

VALUES (значение1, значение2, ...);
```
                  
Здесь: 

+ INSERT INTO— указывает, в какую таблицу добавляем данные
+ В скобках — названия колонок
+ В VALUES — что мы добавляем в каждую колонку

**Пример из интернет-магазина:**

У нас есть таблица `orders` – хранятся заказы пользователей:

|id|	user_id|	total_price|	status|
|:--:|:--:|:--:|:--:|
|1|	101|	2990|	new|
|2|	102|	4900|	shipped|

**Чтобы добавить новый заказ, нужно сделать запрос:**
``` sql
INSERT INTO orders (user_id, total_price, status)

VALUES (103, 5990, 'new');
```
                  
В базу добавится заказ от пользователя с ID = 103, на сумму 5990, со статусом new.

 

**Когда нужны кавычки в SQL в блоке VALUES?**

В SQL в команде `INSERT INTO ... VALUES (...)` мы добавляем данные в таблицу, и разные типы данных записываются по-разному:

|Тип данных|	Пример значения|	Нужно ли ставить кавычки?|
|:--:|:--:|:--:|
|Число (INT)|	990, 1, 103|	Нет|
|Текст (VARCHAR)|	'Бейсболка', 'new', 'аксессуары'	|Да|
|Дата (DATE)|	'2025-01-01'	|Да|
