# DELETE

Запрос `DELETE` в SQL используется для удаления одной или нескольких строк из таблицы базы данных.

Например: пользователь удалил свой заказ --> нужно убрать его из таблицы `orders`.


**Синтаксис:**
``` SQL
DELETE FROM имя_таблицы

WHERE условие;
```
                  
Здесь: 

+ `DELETE FROM` – указываем таблицу, из которой удаляем
+ `WHERE` – пишем, какие строки нужно удалить
 
## ⚠️ Осторожно:

**Если забыть WHERE, удалятся все строки!**
``` sql
DELETE FROM orders; -- Удалит все заказы!
```

**Пример: интернет-магазин**

У нас есть таблица `orders`:

|id|	user_id|	total_price|	status|
|:--:|:--:|:--:|:--:|
|1|	101|	2990|	new|
|2	|102|	4900|	canceled|
|3|	103|	1500|	shipped|

Если заказ отменён, и мы хотим его удалить из базы:
``` sql
DELETE FROM orders WHERE status = 'canceled';
```
                  
Теперь строка с `id = 2` исчезнет из таблицы.
