# UPDATE

Запрос `UPDATE` используется в SQL, чтобы **изменить существующие данные** в таблице.
Например, если пользователь сменил имя или у заказа изменился статус – мы обновляем строку с помощью `UPDATE`.


**Синтаксис:**
``` sql
UPDATE имя_таблицы

SET колонка1 = новое_значение1, колонка2 = новое_значение2

WHERE условие;
```
                  
Здесь:

+ `UPDATE` – указываем, в какой таблице хотим изменить данные.
+ `SET` – какие колонки и на какие значения нужно заменить.
+ `WHERE` – условие, по которому выбирается строка(и) для обновления. 
+ **Без WHERE – обновятся все строки!**
 

**Пример из интернет-магазина:**

У нас есть таблица `orders` – здесь хранятся заказы:

|id|	user_id|	total_price|	status|
|:--:|:--:|:--:|:--:|
|1|	101	|2990|	new|
|2|	102|	4900|	shipped|
|3|	103|	5990|	new|
 

**Задача:** заказ с `id = 3` был доставлен. Нужно изменить статус на `delivered`.
``` SQL
ВЫПОЛНЯЕМ ЗАПРОС: UPDATE orders SET status = 'delivered' WHERE id = 3
```
                  
Теперь заказ id = 3 будет выглядеть так:

|id|	user_id|	total_price|	status|
|:--:|:--:|:--:|:--:|
|1|	101|2990|	new|
|2|	102|	4900|	shipped|
|3	|103|	5990|	delivered|

## ⚠️ Важное замечание:

Если не написать **`WHERE`**, то обновятся **все строки**:
``` SQL
ВЫПОЛНЯЕМ ЗАПРОС: UPDATE orders SET status = 'cancelled';
```
                  

Это сделает **все статусы** в таблице `cancelled`. Будьте осторожны.

|id	|user_id|	total_price|	status|
|:--:|:--:|:--:|:--:|
|1|	101	|2990	|cancelled|
|2|	102|	4900|	cancelled|
|3	|103	|5990	|cancelled|
